## v2.1.1 (2026-03-28) — Блочный парсер расширения

### ✨ Extension: переработка парсинга (задача 1.5)
- Полная переработка `hh.ru.js` → блочный парсер с 9 структурированными блоками
- Новый модуль `resumeExpander.js` — автоматическое разворачивание скрытых блоков (контакты, сопроводительное, «О себе»)
- Формирование `cv_structured` JSON со всеми 9 блоками + метаданные (source, parsed_at, parser_version)
- Формирование `cv_text` — структурированный текст с заголовками блоков для AI (6 блоков)
- Обновлён `api.js` — передача `cv_structured` в POST /api/analyze/
- Обновлён `content.js` — интеграция expander + передача cvStructured в analyze()
- Обновлён `manifest.json` — добавлен resumeExpander.js, версия 2.0.0
- Обратная совместимость: `extractAll()` возвращает и старые поля, и новый `cvStructured`
- Документация расширения: `project_docs/parts/extension_description.md` (задача 1.6)

## v2.1.0 (2026-03-28) — Рефакторинг

### 🐛 Исправлено (8 багов)
- SECRET_KEY регенерация при перезапуске → стабильный default
- Pydantic ValidationError на nullable JSON → все поля Optional с defaults
- Конфликт `format` (date-fns vs query param) → `export_format`
- XSS через v-html → sanitizeHtml()
- Захардкоженные credentials в popup расширения
- Неверный путь `/payments/history` → `/tariffs/payments/history`
- Пустой `.env` → заполнен
- Deprecated `regex` → `pattern` в Query

- Статусы вакансий: active / completed / archived + кнопки перехода

- Даты подбора (начало / окончание)
- Табы в карточке вакансии: Обзор / Кандидаты / Шаблоны
- Клонирование промптов (включая системный → пользовательский)
- Группировка кнопок: информация | активные | опасные
- Шкала оценки 1-100% + 5 уровней рекомендаций
- Экспертный системный промпт v3
- Названия анализов: «Фамилия Имя [МесДД: №N]»
- Ссылка на резюме кандидата в деталях анализа
- Toast-уведомления во всех CRUD-операциях
- Мобильное бургер-меню
- Пользовательский фон (загрузка картинки + кремовый оверлей)
- Extension: «Запомнить меня», без «(любая)»
- Extension: ссылка «На сайте» + confirm «Новый анализ»
- Extension: индикатор авторизации (auth dot)
- Extension: последние 5 анализов в popup
