## Правила работы с документацией

1. **Новый чат с AI** → начинать с `AI_DEV_GUIDE.md` + `INDEX.md`
2. **Новая задача** → проверить `BACKLOG.md` → найти связанные доки через `INDEX.md`
3. **После завершения задачи** → обновить `CHANGELOG.md` + `BACKLOG.md`
4. **Документация устарела?** → пометить `⚠️ OUTDATED` в начале файла + обновить
5. **Не создавать документацию без регистрации в INDEX.md**
